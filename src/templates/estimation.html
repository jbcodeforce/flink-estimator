Flink SQL Queries Complexity

Categorize your Flink SQL queries by complexity. This helps estimate processing overhead.

Simple SQL Queries

SELECT, basic WHERE clauses, simple projections

Examples:
  • SELECT id, name FROM users
  • WHERE status = 'active'
  • Basic field filtering

Medium SQL Queries

JOINs, GROUP BY, basic window operations, aggregations

Examples:
  • Inner/Outer JOINs
  • GROUP BY with aggregations
  • TUMBLE/HOP windows
  • COUNT, SUM, AVG functions

Complex SQL Queries

Advanced analytics, complex windows, nested queries, ML operations

Examples:
  • Complex event pattern matching
  • Multi-level window operations
  • User-defined functions (UDFs)
  • Advanced analytical queries
